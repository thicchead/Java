FORMATTING

% + c (character) + d (int, long)+ f (float) + n (none) + s (any) + S (any)
--> uni char              int	    dec num   newline    string      hoofdletter string

%20d --> 20 plaatsen geven, rechts printen
%-20d --> 20 plaatsen geven, links printen
%020d --> 20 plaatsen, rechts allemaal nullen
%.2f --> afronden op 2 getallen na de komma

constructors --> van kleine tot grote gaan
main constructor gebruikt setExample(example)
	andere gebruiken this(waarde, waarde1)
